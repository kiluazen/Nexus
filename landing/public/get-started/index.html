Add Nexus to ChatGPTClaude

  1. Open Claude's connector settings

    claude.ai → Add custom connector

  2. Paste these two fields, then click Add

    Name
    Nexus
    Remote MCP server URL
    https://claude-mcp.nexus.kushalsm.com/mcp

  3. Start logging

    Say something like "I just did bench press, 3 sets of 60kg for 8." Nexus logs it and you can ask about your history any time.

  1. Open ChatGPT in a desktop browser

    The setup lives on chatgpt.com. Sign in, then open Settings.

  2. Turn on Developer mode

    In Settings, select Security and login and enable Developer mode (one-time).

  3. Create Nexus from Plugins

    Select Plugins in the left sidebar, then click the + button. Name it Nexus, leave authentication set to OAuth, then fill in these two fields:

    Connection Server URL Tunnel
    https://mcp.nexus.kushalsm.com/mcp
    Description
    Nexus lets users log workouts, meals, and body weight from ChatGPT, review their fitness history, correct logged entries, and share progress with accepted Nexus friends.

    Check the custom-MCP risk acknowledgment, click Create, then Sign in with Nexus when ChatGPT prompts you.

  4. Start logging

    Say something like "I just did bench press, 3 sets of 60kg for 8." Nexus logs it and you can ask about your history any time.
