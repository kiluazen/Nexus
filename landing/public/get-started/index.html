Add Nexus to ChatGPT

Nexus is in review for the ChatGPT app directory. Until it's live, you can add it yourself as a connector in about two minutes. Watch the walkthrough, then follow the steps.

  1. Turn on developer mode

    In ChatGPT, open Settings → Connectors → Advanced and enable Developer mode.

  2. Create a new connector

    Back in Connectors, choose Create and name it Nexus.

  3. Paste the Nexus MCP URL

    Use this as the connector's server URL:

    https://mcp.nexus.kushalsm.com/mcp

  4. Sign in

    Complete the Nexus OAuth flow when prompted to link your account.

  5. Start logging

    Say something like "I just did bench press, 3 sets of 60kg for 8." Nexus logs it and you can ask about your history any time.
