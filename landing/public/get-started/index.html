Add Nexus to ChatGPT

  1. Open ChatGPT in a desktop browser

    The setup lives on chatgpt.com. Sign in, then open Settings.

  2. Turn on Developer mode

    In Settings, select Security and login and enable Developer mode (one-time).

  3. Create Nexus from Plugins

    Select Plugins in the left sidebar, then click the + button. Name it Nexus, leave authentication set to OAuth, and paste this Server URL:

    https://mcp.nexus.kushalsm.com/mcp

    Check the custom-MCP risk acknowledgment, click Create, then Sign in with Nexus when ChatGPT prompts you.

  4. Start logging

    Say something like "I just did bench press, 3 sets of 60kg for 8." Nexus logs it and you can ask about your history any time.
