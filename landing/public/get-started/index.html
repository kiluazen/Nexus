Add Nexus to ChatGPT

  1. Open chatgpt.com in a browser

    Custom connectors aren't available in the ChatGPT iOS app — use a desktop or mobile browser.

  2. Create the app

    In ChatGPT, open Settings → Apps → Advanced settings and turn on Developer mode (one-time). Go back, choose Create app, name it Nexus, and paste this as the MCP Server URL:

    https://mcp.nexus.kushalsm.com/mcp

    Check the risk acknowledgment box, hit Create, then Sign in with Nexus when ChatGPT prompts you.

  3. Start logging

    Say something like "I just did bench press, 3 sets of 60kg for 8." Nexus logs it and you can ask about your history any time.
